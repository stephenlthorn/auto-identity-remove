    ⚡ Confirm live action?

    This drives a real browser and submits opt-out requests to live data-broker sites using your configured PII. This is not a drill.

    Welcome - let's set you up

    A few quick questions. Your answers are saved only to this machine's config.json (never sent anywhere else). You can change everything later in the Config tab.

      Who are we removing?

      Data brokers list people by name and contact details. We need your real name and an email so the opt-out requests can identify and reach you.

      Where are you?

      Some brokers are US-only and the request wording (CCPA vs GDPR) depends on your country. City/state/ZIP help match the right person on listing pages.

      CAPTCHA solving - optional, skip for now

      A few broker sites show a CAPTCHA. A CapSolver key lets the tool solve them automatically. You can skip this and add it later in the Config tab - those brokers will just be logged for manual handling.

      Sending opt-out emails - optional

      Some brokers only accept removal requests by email. With SMTP set up, the tool sends those for you; without it they are logged as "manual". Gmail: host smtp.gmail.com, port 587, use an App Password.

      Notifications - optional

      Get a ping when a run finishes. A webhook works on any OS (ntfy.sh, Slack, Discord). iMessage texting is macOS-only. Both are optional.
